# keeper_secrets_manager CHANGELOG

This file is used to list changes made in each version of the keeper_secrets_manager cookbook.

## 0.1.0

Initial release.

- Added `ksm_install` custom resource for installing Keeper Secrets Manager
- Added `ksm_fetch` custom resource for retrieving secrets from Keeper vault
- Support for encrypted data bags, environment variables, and input file authentication
- Cross-platform support (Linux, macOS, Windows)
- Comprehensive test suite with Python unit tests, ChefSpec, and integration tests
